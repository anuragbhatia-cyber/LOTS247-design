# Typography Configuration

## Google Fonts Import

Add to your HTML `<head>`:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=DM+Sans:ital,opsz,wght@0,9..40,100..1000;1,9..40,100..1000&family=IBM+Plex+Mono:wght@400;500;600&display=swap" rel="stylesheet">
```

## Font Usage

- **Headings:** DM Sans (semibold/bold weights)
- **Body text:** DM Sans (regular/medium weights)
- **Code/technical:** IBM Plex Mono (monospaced, for IDs, reference numbers, code)

## Tailwind Classes

```
font-sans  → DM Sans (set as default sans-serif)
font-mono  → IBM Plex Mono
```

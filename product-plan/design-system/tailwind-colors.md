# Tailwind Color Configuration

## Color Choices

- **Primary:** `emerald` — Used for buttons, links, active states, success indicators
- **Secondary:** `amber` — Used for warnings, alerts, tags, secondary highlights
- **Neutral:** `stone` — Used for backgrounds, text, borders, cards

## Usage Examples

### Primary (Emerald)
- Primary button: `bg-emerald-600 hover:bg-emerald-700 text-white`
- Active nav: `bg-emerald-950/60 text-emerald-400`
- Success badge: `bg-emerald-100 text-emerald-800 dark:bg-emerald-900/40 dark:text-emerald-400`
- Link: `text-emerald-600 hover:text-emerald-700 dark:text-emerald-400`

### Secondary (Amber)
- Warning badge: `bg-amber-100 text-amber-800 dark:bg-amber-900/50 dark:text-amber-400`
- Alert icon: `text-amber-500`
- Expiry warning: `text-amber-600 dark:text-amber-400`

### Neutral (Stone)
- Page background: `bg-stone-50 dark:bg-stone-950`
- Card: `bg-white dark:bg-stone-900 border-stone-200 dark:border-stone-800`
- Primary text: `text-stone-900 dark:text-stone-100`
- Secondary text: `text-stone-500 dark:text-stone-400`
- Sidebar: `bg-stone-900` with `text-stone-400` nav items

### Status Colors
- Healthy/Active: `text-emerald-600` / `bg-emerald-100`
- Warning/Expiring: `text-amber-600` / `bg-amber-100`
- Critical/Expired: `text-red-600` / `bg-red-100`
- Info: `text-blue-600` / `bg-blue-100`

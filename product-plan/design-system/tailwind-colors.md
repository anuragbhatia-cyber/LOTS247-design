# Tailwind Color Configuration

## Color Choices

- **Primary:** `emerald` — Used for buttons, links, active states, success indicators
- **Secondary:** `amber` — Used for alerts, warnings, urgency badges, highlights
- **Neutral:** `stone` — Used for backgrounds, text, borders, cards

## Usage Examples

Primary button: `bg-emerald-600 hover:bg-emerald-700 text-white`
Primary badge: `bg-emerald-100 text-emerald-800 dark:bg-emerald-900/30 dark:text-emerald-400`
Secondary badge: `bg-amber-100 text-amber-800 dark:bg-amber-900/30 dark:text-amber-400`
Danger/Error: `bg-red-100 text-red-800` or `text-red-600`
Neutral text: `text-stone-600 dark:text-stone-400`
Neutral border: `border-stone-200 dark:border-stone-700`
Card background: `bg-white dark:bg-stone-900`
Page background: `bg-stone-50 dark:bg-stone-950`
